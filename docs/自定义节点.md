# 自定义节点

## 概述

每个节点双击后会弹出控制面板，面板内容由节点的 `Property` 属性描述符自动生成。以下说明如何自定义节点的属性面板。

---

## 1. 基础的 Property 定义

在节点类中定义 `Property` 类属性即可自动出现在面板中：

```python
from core.node_base import OpenCVNodeDataBase, Property, PropertyGroupNames

class MyFilter(OpenCVNodeDataBase):
    __group__ = "滤波模块"

    # 基本参数
    ksize = Property(5, name="核大小", group=PropertyGroupNames.RUN_PARAMETERS,
                     description="卷积核的尺寸")
    threshold = Property(128, name="阈值", group=PropertyGroupNames.RUN_PARAMETERS,
                         min_val=0, max_val=255)

    # 只读结果参数
    result_count = Property(0, name="检测数量", group=PropertyGroupNames.RESULT_PARAMETERS,
                            readonly=True)

    def __init__(self):
        super().__init__()
        self.name = "自定义滤波"
```

### Property 参数说明

| 参数 | 类型 | 说明 |
|------|------|------|
| `default` | Any | 默认值 |
| `name` | str | 面板中显示的标签 |
| `group` | str | 分组名（决定在哪个标签页显示） |
| `description` | str | 鼠标悬停提示 |
| `readonly` | bool | 只读显示 |
| `order` | int | 同组内排序 |
| `editor` | str | 自定义编辑器类型（见第3节） |
| `choices` | list | 下拉选项列表 |
| `min_val` | number | 最小值约束 |
| `max_val` | number | 最大值约束 |
| `step` | float | 步进值 |
| `decimals` | int | 小数位数 |

### 内部分组名

```python
class PropertyGroupNames:
    RUN_PARAMETERS = "运行参数"      # 主标签页
    BASE_PARAMETERS = "基本参数"
    RESULT_PARAMETERS = "结果参数"
    FLOW_PARAMETERS = "流程控制"
    DISPLAY_PARAMETERS = "显示参数"
    OTHER_PARAMETERS = "其他参数"
```

## 2. 属性值的存取

`Property` 是 Python 描述符，直接通过 `self.xxx` 读写：

```python
def invoke_core(self, src, from_node, diagram):
    k = self.ksize          # 读取
    result = cv2.GaussianBlur(src, (k, k), 0)
    self.result_count = 1   # 写入
    return self.ok(result)
```

## 3. 自定义编辑器类型

通过 `Property` 的 `editor` 参数指定特殊编辑器：

### 3.1 内置编辑器

```python
# 滑块（带数字输入框）
blur = Property(5, name="模糊度", editor="slider", min_val=0, max_val=100)

# 颜色选择器
stroke = Property("#FF0000", name="线条颜色", editor="color")

# 多文件集合
paths = Property([], name="文件列表", editor="file_collection")

# ROI 区域选择
roi = Property(None, name="检测区域", editor="roi")

# 下拉选项
mode = Property("fast", name="模式", editor="choices",
                choices=["fast", "precise", "balanced"])
```

### 3.2 注册自定义编辑器

```python
from PyQt5.QtWidgets import QWidget, QHBoxLayout, QPushButton, QLabel
from gui.property_panel import editor_registry, register_editor

@register_editor("my_button")
def _create_my_button_editor(parent, prop_name, prop_desc, current_value):
    """参数: parent(QWidget), prop_name(str), prop_desc(Property), current_value"""
    container = QWidget(parent)
    layout = QHBoxLayout(container)
    layout.setContentsMargins(0, 0, 0, 0)

    btn = QPushButton(f"执行: {prop_desc.name}")
    label = QLabel(str(current_value))

    def on_click():
        # 处理逻辑
        label.setText("已执行")

    btn.clicked.connect(on_click)
    layout.addWidget(btn)
    layout.addWidget(label)
    return container

# 使用
class MyNode(OpenCVNodeDataBase):
    trigger = Property(False, name="手动触发", editor="my_button")
```

## 4. 自定义属性面板（完全替换）

用 `get_property_presenter()` 返回自定义对象来完全接管面板：

```python
from core.node_base import OpenCVNodeDataBase

class CustomSettings:
    """自定义设置对象——它的 Property 属性会替代原节点的属性面板。"""
    title = "高级设置"

    option_a = Property(True, name="启用A", group="高级")
    option_b = Property(1.0, name="参数B", group="高级", step=0.1)

class MyAdvancedNode(OpenCVNodeDataBase):
    def __init__(self):
        super().__init__()
        self.name = "高级节点"
        self._settings = CustomSettings()

    def get_property_presenter(self):
        return self._settings  # 面板显示 CustomSettings 的属性
```

## 5. 完整示例

```python
"""自定义高斯增强节点"""
import cv2
from core.node_base import OpenCVNodeDataBase, Property, PropertyGroupNames
from core.data_packet import FlowableResult

class EnhancedBlur(OpenCVNodeDataBase):
    __group__ = "滤波模块"

    # ── 运行参数 ──
    ksize = Property(5, name="核大小", group=PropertyGroupNames.RUN_PARAMETERS,
                     min_val=1, max_val=31, step=2)
    sigma = Property(1.0, name="Sigma", group=PropertyGroupNames.RUN_PARAMETERS,
                     min_val=0.0, max_val=10.0, step=0.1, decimals=1)
    mode = Property("blur", name="模糊模式", group=PropertyGroupNames.RUN_PARAMETERS,
                    choices=["blur", "median", "bilateral"])

    # ── 结果参数 ──
    elapsed_ms = Property(0.0, name="耗时(ms)", group=PropertyGroupNames.RESULT_PARAMETERS,
                          readonly=True)

    def __init__(self):
        super().__init__()
        self.name = "高斯增强"

    def invoke_core(self, src, from_node, diagram):
        import time
        t0 = time.time()
        k = self.ksize
        if k % 2 == 0:
            k += 1  # 确保核大小为奇数

        if self.mode == "blur":
            result = cv2.GaussianBlur(src, (k, k), self.sigma)
        elif self.mode == "median":
            result = cv2.medianBlur(src, k)
        else:
            result = cv2.bilateralFilter(src, k, self.sigma * 30, self.sigma * 10)

        self.elapsed_ms = round((time.time() - t0) * 1000, 1)
        return self.ok(result)
```

将此文件放到 `nodes/blurs/` 目录下，程序启动时自动发现并注册。

---

## 6. 面板生命周期

- **打开**：双击节点 → `open_node_dialog()` → 创建 `PropertyPanel` → 调用 `set_node(presenter)` → 读取 `get_property_descriptors()` → 生成表单
- **修改**：用户在面板中修改值 → 实时同步到节点对象（`property.fset`）
- **刷新**：面板每 500ms 自动调用 `_do_refresh()` 同步只读属性（如 `elapsed_ms`）
- **关闭**：点击"关闭"按钮 → 对话框销毁 → 节点保留当前属性值

---

## 7. 注册与发现

节点类只需要：
1. 继承 `OpenCVNodeDataBase`（或其它 `NodeBase` 子类）
2. 设置 `__group__` 类属性指定分组
3. 文件放在 `nodes/` 目录下

程序通过 `node_group.discover_module()` 自动扫描 `nodes/` 下所有模块并注册。

---

## 8. 常见问题

**Q: 修改 Property 默认值不生效？**
A: 在 `__init__` 中设置初始值时，用 `self.xxx = value` 而非直接赋值给类属性。

**Q: 面板值修改后节点没有反应？**
A: Property 描述符会自动触发变更通知。如果需要额外逻辑，在 `__init__` 后手动监听或在 `invoke_core` 中读取最新值。

**Q: 如何在面板中隐藏某个属性？**
A: 不将其定义为 `Property` 类属性，改为普通实例属性即可。

**Q: 如何让属性在特定条件下只读？**
A: 重写 `property_panel.py` 中 `PropertyPanel._is_readonly()` 方法，或使用 `editor_registry` 自定义编辑器。

## 9. 实战示例：文件夹图像源节点

将"图像文件"节点改造为"文件夹图像源"——选择文件夹后自动显示第一张图片，面板内展示缩略图列表，点击缩略图切换显示。

### 9.1 自定义文件夹选择编辑器

```python
# nodes/sources/folder_image_source.py（文件顶部）

import os
import cv2
import numpy as np
from PyQt5.QtWidgets import (QWidget, QVBoxLayout, QHBoxLayout, QPushButton,
                              QLabel, QScrollArea, QFileDialog, QLineEdit,
                              QListWidget, QListWidgetItem)
from PyQt5.QtCore import Qt, QSize
from PyQt5.QtGui import QPixmap, QImage

from core.node_base import (OpenCVNodeDataBase, SrcFilesVisionNodeData,
                            Property, PropertyGroupNames)
from core.data_packet import FlowableResult
from gui.property_panel import register_editor


@register_editor("folder_picker")
def _create_folder_picker_editor(parent, prop_name, prop_desc, current_value):
    """
    文件夹选择 + 缩略图列表编辑器。

    布局：
      [路径输入框] [选择文件夹...]
      [缩略图列表 - 可点击切换]
    """
    container = QWidget(parent)
    layout = QVBoxLayout(container)
    layout.setContentsMargins(0, 0, 0, 0)
    layout.setSpacing(4)

    # ── 第一行：路径 + 按钮 ──
    row = QHBoxLayout()
    row.setContentsMargins(0, 0, 0, 0)

    path_edit = QLineEdit(str(current_value or ""))
    path_edit.setReadOnly(True)
    path_edit.setStyleSheet(
        "QLineEdit { background: #3c3c3c; color: #dcdcdc; border: 1px solid #555; "
        "padding: 3px 6px; font-size: 11px; }")
    row.addWidget(path_edit, 1)

    browse_btn = QPushButton("选择文件夹...")
    browse_btn.setFixedHeight(24)
    browse_btn.setStyleSheet(
        "QPushButton { background: #0078d4; color: white; border: none; "
        "border-radius: 2px; padding: 2px 12px; font-size: 11px; }"
        "QPushButton:hover { background: #1a8ae8; }")
    row.addWidget(browse_btn)

    layout.addLayout(row)

    # ── 第二行：缩略图列表 ──
    thumb_scroll = QScrollArea()
    thumb_scroll.setFixedHeight(86)
    thumb_scroll.setWidgetResizable(True)
    thumb_scroll.setStyleSheet(
        "QScrollArea { background: #1e1e1e; border: 1px solid #3f3f46; }")

    thumb_list = QListWidget()
    thumb_list.setFlow(QListWidget.LeftToRight)
    thumb_list.setViewMode(QListWidget.IconMode)
    thumb_list.setIconSize(QSize(64, 64))
    thumb_list.setSpacing(4)
    thumb_list.setResizeMode(QListWidget.Adjust)
    thumb_list.setStyleSheet(
        "QListWidget { background: #1e1e1e; border: none; }"
        "QListWidget::item { background: #2d2d30; border: 2px solid #3f3f46; "
        "  border-radius: 2px; padding: 2px; }"
        "QListWidget::item:selected { border-color: #0078d4; background: #094771; }"
        "QListWidget::item:hover { border-color: #0078d4; }")
    thumb_scroll.setWidget(thumb_list)
    layout.addWidget(thumb_scroll)

    # ── 生成缩略图的函数 ──
    def _load_thumbs(folder: str):
        """扫描文件夹中的所有图片，加载缩略图到列表。"""
        thumb_list.clear()
        if not folder or not os.path.isdir(folder):
            return

        exts = (".png", ".jpg", ".jpeg", ".bmp", ".tiff", ".tif", ".webp")
        files = sorted(
            f for f in os.listdir(folder) if f.lower().endswith(exts))
        for fn in files:
            fp = os.path.join(folder, fn)
            try:
                img = cv2.imread(fp, cv2.IMREAD_COLOR)
                if img is None:
                    continue
                h, w = img.shape[:2]
                # 缩放到 64×64
                scale = min(64 / w, 64 / h)
                new_w, new_h = int(w * scale), int(h * scale)
                img = cv2.resize(img, (new_w, new_h))
                rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
                h, w, ch = rgb.shape
                qimg = QImage(rgb.data, w, h, w * ch, QImage.Format_RGB888)
                pixmap = QPixmap.fromImage(qimg)

                item = QListWidgetItem()
                item.setIcon(QIcon(pixmap))
                item.setText(fn)
                item.setData(Qt.UserRole, fp)  # 存储完整路径
                item.setSizeHint(QSize(80, 90))
                thumb_list.addItem(item)
            except Exception:
                continue

    # ── 文件夹选择 ──
    def _pick_folder():
        folder = QFileDialog.getExistingDirectory(parent, "选择图像文件夹")
        if folder:
            path_edit.setText(folder)
            parent._set_property_value(prop_name, folder)
            _load_thumbs(folder)

            # 更新节点的文件列表
            node = parent._current_node
            if node and hasattr(node, 'set_folder'):
                node.set_folder(folder)

    browse_btn.clicked.connect(_pick_folder)

    # ── 点击缩略图切换当前图片 ──
    def _on_thumb_clicked(item):
        fp = item.data(Qt.UserRole)
        node = parent._current_node
        if node and fp:
            node.src_file_path = fp
            # 触发主图像区刷新
            if hasattr(parent, '_image_viewer') and parent._image_viewer:
                try:
                    mat = cv2.imread(fp, cv2.IMREAD_COLOR)
                    if mat is not None:
                        parent._image_viewer.set_image(mat)
                except Exception:
                    pass

    thumb_list.itemClicked.connect(_on_thumb_clicked)

    # 初始加载
    if current_value and os.path.isdir(str(current_value)):
        _load_thumbs(str(current_value))

    return container
```

### 9.2 自定义节点类

```python
class FolderImageSourceNode(SrcFilesVisionNodeData, OpenCVNodeDataBase):
    """从文件夹加载图像——自动显示第一张，缩略图列表切换。

    替换原来的 SrcImageFilesNodeData。
    """
    __group__ = "图像数据源"

    # ── 运行参数 ──
    folder_path = Property("", name="文件夹", group=PropertyGroupNames.RUN_PARAMETERS,
                           editor="folder_picker",  # 使用上面注册的自定义编辑器
                           description="选择包含图像的文件夹")
    use_all_image = Property(False, name="使用所有图像",
                             group=PropertyGroupNames.RUN_PARAMETERS)

    def __init__(self):
        SrcFilesVisionNodeData.__init__(self)
        OpenCVNodeDataBase.__init__(self)
        self.name = "文件夹图像源"

    def set_folder(self, folder: str):
        """选择文件夹后：清空旧文件 → 扫描新文件夹 → 选择第一张。"""
        self.clear_files()
        self.add_files_from_folder(folder)
        self.folder_path = folder
        if self.src_file_paths:
            self.src_file_path = self.src_file_paths[0]

    def invoke_core(self, src, from_node, diagram) -> FlowableResult:
        path = self.src_file_path
        if not path or not os.path.exists(path):
            return self.error(None, f"文件不存在: {path}")

        mat = cv2.imread(path, cv2.IMREAD_COLOR)
        if mat is None:
            return self.error(None, f"无法读取: {os.path.basename(path)}")

        self.pixel_width = mat.shape[1]
        self.pixel_height = mat.shape[0]
        return self.ok(mat, f"已加载: {os.path.basename(path)}")

    def to_dict(self) -> dict:
        data = super().to_dict()
        data["folder_path"] = self.folder_path
        return data

    def restore_from_dict(self, data: dict):
        super().restore_from_dict(data)
        fp = data.get("folder_path", "")
        if fp:
            self.folder_path = fp
            self.set_folder(fp)
```

### 9.3 数据流说明

```
用户点击"选择文件夹..."
  → _pick_folder() 弹出 QFileDialog 选择文件夹
  → path_edit 显示路径
  → parent._set_property_value() 写入 node.folder_path
  → node.set_folder(folder)：
      1. clear_files() 清空旧列表
      2. add_files_from_folder(folder) 扫描所有图片
      3. src_file_path = 第一张图片路径
  → _load_thumbs(folder) 生成缩略图列表

用户点击缩略图
  → _on_thumb_clicked(item)
  → node.src_file_path = 选中文件的完整路径
  → 主图像区 ImageViewer 显示该图片
```

### 9.4 关键差异对比

| 特性 | 原始"图像文件" | 改造后"文件夹图像源" |
|------|---------------|---------------------|
| 面板属性 | `src_file_path` = "当前文件" | `folder_path` = "文件夹" |
| 选择器 | `QFileDialog.getOpenFileName` (文件) | `QFileDialog.getExistingDirectory` (文件夹) |
| editor 提示 | 无（自动检测 path/file 关键词） | `editor="folder_picker"` (显式指定) |
| 缩略图 | 无 | 64×64 缩略图列表 |
| 切换方式 | 手动修改路径 | 点击缩略图自动切换 |
